JSON Tagger Documentation

JSON Tagger adds a JSON API on top of Stagger, the part-of-speech tagger developed by Robert Östling from Stockholm University.

Basic usage

curl {{ site }}/api/tag -d "Den skogsbrand som rasade i Västmanland"
